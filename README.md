# MindMate 🌸

MindMate is an AI-powered mental wellness chatbot that provides supportive conversations and mood-based music recommendations.

## 🚀 Live Demo
https://mindmate-pcyo.onrender.com

## ✨ Features
- AI chatbot
- Mental wellness support
- Mood detection
- Music recommendations
- Multi-language support

## 🛠️ Tech Stack
- Node.js
- Express.js
- Groq API
- HTML
- CSS
- JavaScript
